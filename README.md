# OSM-MusicPlayer
###Owais Studio Music(OSM) is online music player which is purely based on HTML and CSS only.

## Functions/Features
   - Home Page
      - Cursor Section
      - Different Section realted songs
        - Latest Songs
        - New Release Songs
        - Mood's Songs
        - Top 50 Songs 
      - Side bar
        - Trending Songs
        - Event & Facts
        - Artists Birthday
      - About & Follow Section
      - Player Bar Footer
   
   - Artists Page
      - Artist details
      - Artists Follow Button
      - Artists Songs
      - Similar Artists

   - Both pages are responsive
